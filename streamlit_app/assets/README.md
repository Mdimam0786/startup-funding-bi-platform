# Assets

This folder stores optional project assets.

- **resume.pdf** *(optional)*  
  Place your résumé in this folder using the filename **`resume.pdf`**. The application will automatically detect it and enable the **Download Résumé** button in the sidebar and on the About page. No code changes are required.

- **images/** *(optional)*  
  Store logos, screenshots, or other image assets here. By default, the application uses a CSS-based wordmark and SVG icons, so no image files are required. If you want to use a custom logo (for example, `logo.png`), add it to this folder and update `components/sidebar.py` to display it.
